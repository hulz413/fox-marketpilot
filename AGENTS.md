# AGENTS.md

## Language

- 面向人类阅读的非结构化内容使用中文，结构化字段和结构化内容使用英文。
- Markdown 普通章节标题使用中文；OpenSpec 等工具要求的解析性标题或前缀（例如 `ADDED Requirements`、`MODIFIED Requirements`、`Requirement`、`Scenario`、`WHEN`、`THEN`、`AND`）可保留英文。
- 除上述解析性格式、代码标识、接口路径、枚举值和命令外，面向人阅读的标题、段落、列表项和任务描述必须使用中文。

## Project Docs

- 产品需求以 `docs/mvp/roadmap.md` 为准，技术栈以 `docs/techstack.md` 为准。

## Git

- Git commit message 必须遵循不带 scope 的 Conventional Commits 格式：`<type>: <description>`。
