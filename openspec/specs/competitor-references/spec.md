## Purpose

定义 MarketPilot 为商机生成、保存、读取和展示竞品或类似产品参考的能力，帮助用户理解市场上已有类似产品、常见售价和差异化空间，同时保持待验证和谨慎表达。

## Requirements

### Requirement: 系统持久化竞品参考

系统 SHALL 为每个已生成商机保存可读取的结构化竞品参考，并与研究任务、商机和可用来源建立稳定关联。

#### Scenario: 保存竞品参考基础字段

- **WHEN** 系统成功为商机生成竞品参考
- **THEN** 系统保存竞品参考 UUID
- **AND** 系统将竞品参考关联到对应研究任务和商机
- **AND** 每个竞品参考包含排序、参考名称、参考市场、售价区间、常见卖点、同质化程度、差异化切入点、参考说明和来源状态
- **AND** 竞品参考包含创建时间、更新时间和可为空的删除时间
- **AND** 系统不对外暴露竞品参考、研究任务、商机或来源的内部自增 ID

#### Scenario: 每个商机保存多个竞品参考

- **WHEN** 系统为一条研究任务生成竞品参考
- **THEN** 每个未软删除商机至少保存 2 个竞品或类似产品参考
- **AND** 同一商机下的竞品参考排序不重复
- **AND** 排序可用于商机详情和报告摘要展示

#### Scenario: 关联竞品来源

- **WHEN** 竞品参考使用了已收集研究来源
- **THEN** 系统将竞品参考关联到一个或多个未软删除来源
- **AND** 关联来源优先来自同一商机的 `competitor` 或 `general` 来源
- **AND** 每个来源关联包含可读的关联说明
- **AND** 来源关联不表示竞品、销量、售价或市场规模已经被全面核验

#### Scenario: 重新运行替换旧竞品参考

- **WHEN** 一条已完成研究任务被重新运行并成功生成新商机和新竞品参考
- **THEN** 系统将该任务上一轮竞品参考从默认读取中排除
- **AND** 系统保存新运行生成的竞品参考
- **AND** 后续读取该任务竞品参考时默认返回最新未软删除结果

### Requirement: 系统生成竞品参考

系统 SHALL 在基础商机、来源线索、需求洞察和货源候选阶段结束后，为当前研究任务的每个未软删除商机生成竞品参考。

#### Scenario: 基于商机和来源生成竞品参考

- **WHEN** 基础商机结果已经保存且来源收集阶段已经结束
- **THEN** 系统读取当前任务的未软删除商机
- **AND** 系统读取当前任务的未软删除来源
- **AND** 系统为每个商机生成至少 2 个竞品或类似产品参考
- **AND** 竞品参考说明类似产品示例、常见售价区间、常见卖点、同质化程度和差异化切入点
- **AND** 关键判断尽量关联可用来源

#### Scenario: 缺少竞品来源时生成待验证参考

- **WHEN** 某个商机没有可用的 `competitor` 或 `general` 来源
- **THEN** 系统仍可以基于研究任务输入、商机字段、需求洞察或货源候选生成待验证竞品参考
- **AND** 系统将来源状态标记为缺少来源或 fallback
- **AND** 参考文案说明售价、卖点、同质化程度和差异化空间需要继续通过公开资料或小批量验证确认

#### Scenario: 竞品参考输出保持谨慎表达

- **WHEN** 系统生成参考名称、参考说明、常见卖点、差异化切入点或来源关联说明
- **THEN** 文案使用“类似产品参考”“公开线索”“可能”“需要验证”“待确认”或等价谨慎表达
- **AND** 文案不使用“竞品已全面核验”“售价已确认”“销量已确认”“市场已证明”或等价表达

#### Scenario: 本地或测试环境不依赖外部服务

- **WHEN** 系统运行在 local 或 test 环境且未配置 LLM API key
- **THEN** 系统使用确定性 fallback 生成中文竞品参考
- **AND** fallback 结果仍满足竞品参考 schema
- **AND** 自动化测试不依赖外部 LLM、搜索或网络服务

### Requirement: 用户可以读取竞品参考

系统 SHALL 允许前端读取某条研究任务或单个商机的竞品参考，用于商机详情和基础报告展示。

#### Scenario: 读取任务竞品参考列表

- **WHEN** 用户请求一条存在且未软删除研究任务的竞品参考列表
- **THEN** 系统返回该任务未软删除商机关联的未软删除竞品参考
- **AND** 每条竞品参考包含竞品参考 UUID、研究任务 UUID、商机 UUID、排序、参考名称、参考市场、售价区间、常见卖点、同质化程度、差异化切入点、参考说明、来源状态和关联来源摘要
- **AND** 结果按商机推荐排序和竞品参考排序稳定排列
- **AND** 系统不返回内部自增 ID

#### Scenario: 读取单个商机竞品参考

- **WHEN** 用户请求一个存在且未软删除商机的竞品参考列表
- **THEN** 系统返回该商机未软删除的竞品参考
- **AND** 返回内容包含关联来源摘要
- **AND** 系统不返回其他商机的竞品参考

#### Scenario: 读取没有竞品参考的任务或商机

- **WHEN** 用户请求的任务或商机尚未生成竞品参考
- **THEN** API 响应保持成功
- **AND** 系统返回空列表
- **AND** 前端可以展示中文非阻塞空状态

#### Scenario: 读取不存在资源

- **WHEN** 用户请求不存在或已软删除的研究任务或商机
- **THEN** 系统返回未找到错误
- **AND** 系统不返回其他任务或商机的数据

### Requirement: 前端展示竞品参考

前端 SHALL 在商机详情和基础报告中展示竞品参考，帮助用户理解类似产品现状和差异化空间。

#### Scenario: 商机详情展示完整竞品参考

- **WHEN** 用户打开单个商机详情且该商机存在竞品参考
- **THEN** 页面展示参考名称、参考市场、售价区间、常见卖点、同质化程度、差异化切入点和参考说明
- **AND** 页面展示关联来源入口或来源摘要
- **AND** 页面说明竞品参考仍需通过公开资料或小批量验证确认

#### Scenario: 基础报告展示竞品参考摘要

- **WHEN** 用户打开已完成研究任务的基础报告且该任务存在竞品参考
- **THEN** 页面为每个商机展示竞品参考摘要
- **AND** 页面提供进入单个商机详情查看完整竞品参考的入口
- **AND** 页面不把竞品参考展示为已确认竞品、已核验售价或已证明市场

#### Scenario: 竞品参考为空或读取失败

- **WHEN** 竞品参考接口返回空结果或读取失败
- **THEN** 页面展示中文可理解的暂不可用说明
- **AND** 页面保留基础商机推荐、详情或报告主体内容
- **AND** 页面不将缺少竞品参考展示为任务失败

### Requirement: 竞品参考生成可观测且非阻塞

系统 SHALL 为竞品参考生成记录可观测信息，并确保竞品参考失败不破坏已完成的基础商机结果。

#### Scenario: 记录竞品参考生成阶段

- **WHEN** 后台研究运行执行竞品参考生成
- **THEN** 系统记录 `generate_competitor_references` 阶段事件
- **AND** 阶段事件包含任务 UUID、运行 ID、阶段状态、耗时和可安全展示的错误摘要
- **AND** LangSmith tracing 启用时，该阶段与同一研究运行 trace 关联

#### Scenario: 竞品参考生成失败

- **WHEN** 基础商机结果已经保存但竞品参考生成失败
- **THEN** 研究任务仍可保持 `completed`
- **AND** 系统不删除已生成的商机结果、来源、需求洞察或货源候选
- **AND** 系统记录竞品参考失败的阶段事件和应用日志
- **AND** 竞品参考读取接口返回空结果
- **AND** 用户可见错误摘要不包含敏感凭证、原始异常堆栈或内部自增 ID
